## Contents

- ### About page
  - 자기소개를 비록한 간략한 정보 페이지

  - 여태까지 배우거나 써 본 기술 스택을 보여주는 페이지
- ### Project page
  - #### 팀 프로젝트
    - 팀원들과 함께한 프로젝트에 대한 상세한 설명을 프로젝트 기간, 사용한 기술, 배포 링크, 시연 영상, 프로젝트 레포지토리와 함께 적음.
  - #### 개인 프로젝트
    - 혼자 진행했던 프로젝트에 대한 상세한 설명을 프로젝트 기간, 사용한 기술, 배포 링크, 시연 영상, 프로젝트 레포지토리와 함께 적음.
